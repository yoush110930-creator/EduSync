# EduSync – Writing Optimization Extension for Chrome

EduSync is a Chrome extension designed to help users refine, optimize, and automate text entry directly within their browser.  
It enhances clarity, tone, and structure while providing controlled auto-typing functionality — all without requiring additional tabs or external tools.

---

## 🚀 Key Features

### 1. Academic Optimization (Humanizer)
Refines text to improve readability, flow, and tone while preserving meaning.  
This feature is designed to help users:

- Improve coherence and structure  
- Reduce awkward phrasing  
- Enhance clarity and readability  
- Adjust tone to sound natural and authentic  

**Usage:** Users provide text manually, and the extension assists in refinement.

### 2. Auto-Typing (Transcription Emulator)
A controlled auto-typing tool that types user-provided text into selected fields at adjustable speed and accuracy.

This feature is designed to:

- Automate repetitive typing tasks  
- Save time when entering prepared content  
- Simulate natural typing speed  
- Assist users with accessibility or workflow efficiency  

Users input their own text; EduSync does **not** generate content automatically in this feature.

---

## 📌 Product Terminology

EduSync uses branded terminology for clarity:

- **Academic Optimization** → Humanizer (text refinement)  
- **Auto-Typing** → Transcription Emulator (types user-provided text)  
- **Source Material** → Input field for original text  
- **Processed Output** → Refined or typed result  
- **Units** → Words  
- **Finalize Refinement** → Humanize (Start Button for Humanizer)  
- **Processing Duration** → Typing Time (speed of auto-typing)  
- **Variance Density** → Typographic Error Percentage (accuracy of auto-typing)  

All features are writing assistance tools. Users are responsible for compliance with academic, workplace, or institutional policies.

- Coming soon! Currently, EduSync is available via Developer Mode installation.

### Manual Installation (Developer Mode)
1. Download the ziped extension folder from [Download](DOWNLOAD.md).  
2. Navigate to: `chrome://extensions/`  
3. Enable **Developer Mode** (top-right corner)  
4. Click **Load unpacked**  
5. Select the extension folder

## 🧭 How to Use

### Academic Optimization (Humanizer)
1. Open EduSync from the Chrome toolbar.  
2. Select **Academic Optimization**.  
3. Paste text into the **Source Material** input field.  
4. Click **Finalize Refinement** to humanize the text.  
5. Review the **Processed Output** carefully.  
6. Click **Transfer Data** to copy the result.

### Auto-Typing (Transcription Emulator)
1. Open EduSync from the Chrome toolbar.  
2. Select **Transcription Emulator**.  
3. Click inside the target text field where you want text typed.  
4. Paste or type text into the input field.  
5. Adjust **Typing Time** (speed) and **Variance Density** (accuracy).  
6. Click **Initialize Transcription** to begin auto-typing.  
7. Always review the content before submission or publication.

## 🔍 Troubleshooting

**Extension not responding**  
- Refresh the webpage  
- Verify the extension is installed  
- Restart Chrome  

**Text not processing**  
- Ensure text is selected when using Auto-Typing  
- Confirm the site allows content scripts: `Manage Extensions → EduSync → Enable Site Access`  
- Disable conflicting extensions  

**Processing errors**  
- Check internet connection  
- Confirm any API services are available
